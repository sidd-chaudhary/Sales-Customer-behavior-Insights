# Customer Shopping Behavior Analysis

## Project Overview

This project focuses on analyzing customer shopping behavior using Python, PostgreSQL, SQL, and Power BI. The objective was to clean raw retail data, perform business analysis, and build an interactive dashboard to generate meaningful business insights.

The project demonstrates a complete end-to-end data analytics workflow commonly used in real-world business intelligence projects.



 Project Objectives

* Clean and preprocess customer shopping data using Python and Pandas.
* Store structured data in PostgreSQL database.
* Perform SQL queries for business analysis.
* Build an interactive Power BI dashboard.
* Generate insights related to customer behavior and sales performance.



 Tools & Technologies Used

* Python
* Pandas
* NumPy
* Jupyter Notebook
* PostgreSQL
* SQL
* Power BI



 Dataset Information

The dataset contains customer shopping behavior and retail transaction information, including:

* Customer demographics
* Product categories
* Purchase amounts
* Payment methods
* Review ratings
* Subscription status
* Seasonal sales information

Dataset Summary

* Total Records: 3,900
* Total Columns: 18



 Data Cleaning & Preprocessing

The following preprocessing steps were performed using Python and Pandas:

* Imported and explored the dataset
* Checked missing values
* Handled null values
* Corrected data types
* Created additional analytical columns
* Prepared cleaned dataset for SQL and Power BI integration



 SQL Analysis

SQL queries were used to analyze:

* Revenue by gender
* Top-performing product categories
* Seasonal sales trends
* Subscription customer behavior
* Customer purchasing insights

---

 Power BI Dashboard

An interactive Power BI dashboard was developed to visualize:

* Total Revenue
* Customer Distribution
* Category Performance
* Seasonal Sales Trends
* Payment Method Analysis
* Subscription Insights

---

 Key Business Insights

* Clothing category generated the highest revenue.
* Male customers contributed a larger share of total sales.
* Seasonal trends impacted customer purchasing behavior.
* Subscription customers showed different purchasing patterns.
* Payment methods were used almost equally by customers.
